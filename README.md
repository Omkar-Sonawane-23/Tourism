

# Bharat Yatra

Bharat Yatra is a tourist app that provides descriptions about different tourist destinations in India. This project is built using React and Vite, offering a fast and modern development experience.

## Getting Started

### To contribute to this project, follow these steps:

1. **Star this Repository**: You can find the Star button on top-right corner.

    <img src="https://docs.github.com/assets/cb-40742/images/help/repository/fork-button.png" width="80%"/>

2. **Fork this Repository**: Fork this repository to create your copy on GitHub.It is just to the left of star button.

3. **Clone the Repository**: Clone your forked repository to your local machine using the following command:

    a. Open the forked repository ( It is in your profile repository)>>Go to Green Code dropdown button and copy the link 
    
     <img src='https://docs.github.com/assets/cb-69468/images/help/repository/https-url-clone-cli.png' height="50%" width="80%"/>
    
    b. Open Command Prompt(cmd)/Git Bash and navigate to your desired location, where you want your project folder to be.
    
   ```bash
   git clone <link>
   //The Link should look like https://github.com/<your-username>/Bharat-Yatra.git
   
    ```
    Note: if you have already forked the repository before, you need to update it before working on your issue.
    
    ```bash
        git remote update 
    ```
    c. Navigate to the Project Directory
    
    ```bash
        cd Tourism
    ```
    
    d. Open your Project in your Code Editor. If you use VS Code :
    
    ```bash
        code .
    ```
    
    
### To run this project, follow these steps:
1. Open the terminal in VS Code (Ctrl + Shift + `)
2. Type the following commands:
    
    a. 
    
    ```bash
        npm i 
    ```
    You will observe node modules added to your directory. (Remember we should not push node modules while we are pushing our code to github)

    b. 
    ```bash
        npm run dev
    ```
    This will run your project in http://localhost:5174/. 
    Open your browser to see the Project.
    
    c. Make changes and fix the issue. (Work on it only if you are assigned the issue)

### To push your changes and create PR, follow these steps:

1. Open your VS Code Terminal
2. Create your branch using the following command
    ```bash
        git checkout -b <"branch_name">
        //Replace <branch_name> your barnch name. Try to name it properly like issue_no._name

    ```

3. Check Status

    ```bash
        git status
        //You will get list of modified files. Copy the file names you modified.
    ```
    
4. Add your files. Make sure you are not adding node_modules. 
    ```bash
    git add <filenames>
    //Paste those file names instead of <filenames>.
    ```
5. Commit your changes.
    ```bash
        git commit -m <"good_commit_message">
    ```
6. Push your changes
    ```bash
        git push --set-upstream origin <"branch_name">
    ```
7. Go to the original repository in Github. You will see <b>Compare and Pull Request</b>. Click on it.
    <img src="https://i.stack.imgur.com/9JycO.png" width="80%"/>

8. You should create PR to dev branch. Change main to dev. Add Title and Description.

9. Create PR(Pull Request). Copy the URL.

10. Go to your issue , add the URL of your PR in the comments and ask to approve.
